# 追色工具 — 使用说明

> 数字影像调色专家系统 | Lab 色彩迁移 + 肤色保护 + 专家管线

## 快速启动

双击 **「启动追色工具.command」** 即可打开界面。

依赖：Python 3.14 + numpy + Pillow + scipy + customtkinter + scikit-learn（可选）

## 两种工作模式

### 简化模式 — 一键追色
1. 导入参考图和原图
2. 调整影调/色调强度滑块
3. 点击「▶ 开始追色」

### 专家模式 — 精细控制
1. 导入参考图和原图
2. 点击「🔍 自动分析差异」自动填充参数
3. 微调各参数
4. 或直接在「风格」标签页选择预设
5. 点击「▶ 开始追色」

## 参数说明

### 基础参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 影调强度 | 0～100% | 控制亮暗对比的匹配程度（L通道） |
| 色调强度 | 0～100% | 控制颜色风格的匹配程度（a,b通道） |
| 肤色保护 | 0～100% | 0=不保护，100=皮肤完全不变 |
| 全局强度 | 0～100% | 整体追色效果的强度系数 |

### 进阶参数 (Lightroom 风格)

| 参数 | 范围 | 说明 |
|------|------|------|
| 曝光 | -3～+3 EV | 整体明暗调整 |
| 对比度 | -100～+100 | 影调反差 |
| 高光 | -100～+100 | 亮部区域调整 |
| 阴影 | -100～+100 | 暗部区域调整 |
| 白色 | -100～+100 | 极亮区调整 |
| 黑色 | -100～+100 | 极暗区调整 |
| 分离色调 | 色相0-360° / 饱和0-100 | 高光/阴影分别加色 |
| 色温 | -100～+100 | 蓝(-)→橙(+) |
| 色调 | -100～+100 | 绿(-)→品红(+) |

### 风格预设

| 预设 | 风格 | 特点 |
|------|------|------|
| 🌿 日系清新 | 滨田英明 | 高亮度低对比，微冷偏青白 |
| 🎬 暗调电影 | 王家卫港风 | 低调高反差，冷蓝暗调 |
| 📷 复古胶片 | 胶片质感 | 提黑场，高光偏橙阴影偏绿 |
| 🎭 青橙色调 | 电影级T&O | 暗部偏青亮部偏橙 |
| 🎞 Kodak Portra | 经典人像 | 柔和暖白，肤色通透 |

## 技术原理

- **色彩空间转换**：纯 NumPy 实现 sRGB D65 → XYZ → CIE Lab 标准路径（无 OpenCV 依赖）
- **色彩迁移（简化）**：Reinhard Lab 方法，在 L\*a\*b\* 空间对均值和标准差进行匹配
- **色彩迁移（专家）**：Lightroom 风格完整管线 — 曝光→对比度→高光阴影→RGB曲线→HSL→分离色调
- **肤色保护**：HSV + Lab 双空间检测亚洲人肤色，ΔE>5 自动回滚，保守区仅做提亮+保暖
- **特征提取**：影调分位数 + Lab统计 + K-Means 5色色板 + 肤色锚点 + 环境主色
- **评价指标**：直方图相似度 + 主色色相差 + ΔE + Lab 匹配度
- **LUT 生成**：构造标准颜色网格，经追色引擎映射后写成 .cube 格式

## 支持格式

输入：JPG / PNG / TIFF / BMP / WebP
输出：JPG / PNG / TIFF / .cube LUT
