追色 - AI智能配色工具 v2.0
============================

基于完整色彩工程的专业追色系统

依赖安装
--------
pip install numpy pillow scipy scikit-learn

运行方式
--------
python app.py

功能说明
--------
1. 参考图导入 - 加载目标色调的参考图片
2. 色彩分析 - 提取参考图和原片的色彩特征（专业模式）
3. 智能追色 - Lab空间色彩迁移 + ΔE肤色保护
4. LUT导出 - 导出.cube格式的LUT文件，可在PS/LR中使用
5. 批量处理 - 支持多图批量调色

两种模式
--------
- 快速模式: 直接使用Lab Reinhard色彩迁移
- 专业模式: 完整色彩分析 + ΔE肤色保护 + 分离色调

无OpenCV依赖，纯PIL + NumPy + scipy实现
