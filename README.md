# 追色工具 — 使用说明

## 快速启动

双击 **「启动追色工具.command」** 即可打开界面。

---

## 使用流程

### 基础追色（图片）
1. 点击左侧「参考图」区域，导入你找到的风格参考图
2. 点击中间「原图」区域，导入你要调色的照片
3. 调整右侧滑块：
   - **影调强度**：控制亮暗对比的匹配程度（建议 60~90%）
   - **色调强度**：控制颜色风格的匹配程度（建议 70~100%）
4. 勾选「启用肤色保护」，用**保护强度**控制皮肤颜色保留程度
5. 点击「▶ 开始追色」
6. 满意后点击「↓ 保存结果图」

### 导出 LUT（用于 PR / AE / 达芬奇）
1. 完成参考图导入和参数调整
2. 点击「↓ 导出 .cube LUT」
3. 选择精度（33 标准即可）
4. 在 Premiere Pro 中：效果 → Lumetri 颜色 → 基本校正 → 输入 LUT → 选择导出的 .cube 文件

---

## 参数说明

| 参数 | 范围 | 说明 |
|------|------|------|
| 影调强度 | 0～100% | 0=不调亮暗，100=完全匹配参考图影调（L通道） |
| 色调强度 | 0～100% | 0=不调色彩，100=完全匹配参考图色调（a,b通道） |
| 肤色保护 | 0～100% | 0=不保护，100=皮肤完全不变（YCrCb肤色检测）|
| LUT 精度 | 17/33/65 | 格点越多精度越高，文件越大 |

---

## 技术原理

- **色彩迁移**：Reinhard Lab 方法，在 L*a*b* 颜色空间对均值和标准差进行匹配
- **肤色保护**：YCrCb + HSV 双空间检测肤色区域，生成平滑遮罩，在追色结果与原图之间按比例混合
- **LUT 生成**：构造标准颜色网格，经追色引擎映射后写成 .cube 格式，兼容所有专业软件

---

## 支持格式

输入：JPG / PNG / TIFF / BMP / WebP  
输出：JPG / PNG / TIFF / .cube LUT
